About

The DOI Bookmarklet provides access to access-restricted documents by redirecting through a ProQuest EZproxy as used by many university libraries.

Installation

Enter your ProQuest EZproxy URL:

Bookmark the following link: DOI Bookmarklet

The bookmarklet consists of the following generated and minified JavaScript:

(function(){var a=document.getSelection?document.getSelection().toString():document.selection.createRange().text,a=/\w?(?:doi:|hdl:(?:\/\/)?)\s*([0-9./a-z]+)\w?/i.exec(a);if(null!=a){var a=a[1],b=document.createElement("a");b.href="";console.log(b.protocol+"//dx.doi.org."+b.host+"/"+a)}})();

How it works

The bookmarklet searches the currently selected text for a DOI (beginning with doi: or hdl:) and routes the subsequent request to http://dx.doi.org through the specified proxy.
